Firmware Flashing Guide

For Most Users (Recommended)

If you only need to update or recover your device, follow these steps:

1. Download the appropriate tool for your platform (Windows recommended)
2. Extract the downloaded .zip file
3. Install the required driver (Windows only)
4. Connect the device via USB
5. Follow the step-by-step guide in our official documentation

⸻

Downloads

Windows (Recommended)

* DriverAssistant_v5.14_win.zip
    USB driver for device detection. Must be installed before flashing.
* SocToolKit_v2.5_win.zip
    Official firmware flashing tool for Windows.
